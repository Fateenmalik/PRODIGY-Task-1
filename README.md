# PRODIGY_DS_01

## Task 01: Population Distribution Visualization

### Objective

Create a bar chart or histogram to visualize the distribution of a categorical or continuous variable using real-world data.

### Dataset

World Bank Population Dataset

Source:
https://data.worldbank.org/indicator/SP.POP.TOTL

### Tools and Libraries Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Seaborn

### Project Workflow

1. Imported the World Bank population dataset.
2. Performed data cleaning and preprocessing.
3. Selected the latest available population data.
4. Analyzed population distribution across countries.
5. Created a bar chart showing the top 10 most populated countries.
6. Created a histogram to visualize population distribution.

### Visualizations

* Bar Chart: Top 10 Countries by Population
* Histogram: Distribution of Population Across Countries

### Key Insights

* India and China are the most populated countries.
* Population distribution is highly uneven across countries.
* A small number of countries account for a large share of the world's population.
* Most countries have relatively smaller populations compared to the top-ranked countries.
